What is an IP address?

IP is short for Internet Protocol. An IP address is a numerical label assigned to each device connected to a computer network so that messages can be sent and received in the same format. It looks like this (for now):

The current protocol we are using is called IPv4. It was created in 1973 by the founders of the internet Vint Cerf and Robert E. Khan and it waswidely adopted in the early 1980's. This version uses 32 bits per address, as shown in this picture:

The problem with this version is that it only offers about 4 billion unique addresses and we will run out soon. No one expected the internet to be so popular!

What is IPv6?

The solution is IPv6: a new and longer IP address format. Its development started in 1992 by the Internet Engineering Task Force (IETF) and later deployed to the world in 2012. An IPv6 address looks like this:

It uses 128 bits per address, which creates 340 undecillion unique addresses. This is more than enough for every grain of sand on earth (we're talking beaches AND desserts) to have its own unique IP address.

Implementation / Effects

Change is hard and expensive therefore changing to IPv6 is a multi year transition that will take some patience. These are some of the roadblocks that IPv6 is currently facing:

  1. We want to completely replace IPv4 which takes time to change all the addresses we already have.

  2. IPv4 and IPv6 are not meant to work together which makes the transition also a tad more complicated. Although there has been some things created to allow exchange between the two, this is only a temporary fix.

  3. Internet service providers find it too expensive to switch to solely IPv6 equipment or dual stack gear that can handle both protocols.

However, there are many reasons why no one will give up on IPv6. Aside from providing many more addresses to allow more computers to be connected to the network, IPv6 has many more advantages to IPv4.
